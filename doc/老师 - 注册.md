## 注册页面
* url: /reg
* floder: teacher/site/reg.blade.php

## 执行注册
* url: /doreg
* method: POST
* request: 
```
{
    tel: '手机号码',
    password: '密码'
}
```

* response
```
{
    msg: '注册成功'
}

```
