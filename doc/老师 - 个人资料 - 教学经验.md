## 保存信息
* url: /exp/save
* method: POST
* request:
```
{
    title: "标题,string" 长度2-20
    start_time: "开始时间,string" (2016-11)
    end_time: "结束时间,string"
    description: "描述,string" 长度20-255
}
```
* response
```
* 422错误
 {
    key: [
        '错误信息'
    ]
 }

* 400错误
{
    msg: '错误信息'
}

* 200
{
    msg: 信息提示
}

```
